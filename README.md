## 🎯 Overview

EduBuddy is a comprehensive AI-powered learning platform designed for students in technical sectors. It combines intelligent tutoring, resource management, community collaboration, and gamification to create an engaging educational experience.

Whether you're studying Web Design, Informatique Décisionnel et IA, or Génie Informatique, EduBuddy provides the tools and support you need to succeed.

---

## ✨ Features

### 📚 Smart Resource Management
- **Document Organization** - Organize courses, TDs, TPs, and exams by sector and year
- **Smart Search** - Find resources instantly with powerful search capabilities
- **Download Tracking** - Monitor your study materials and learning progress
- **File Management** - Upload and share study resources with your peers

### 🤖 AI-Powered Assistant
- **24/7 Learning Support** - Get instant answers to your academic questions
- **Smart Explanations** - Complex topics explained in simple terms
- **Code Assistance** - Help with programming problems and concepts
- **Multi-Subject Support** - Math, Programming, Databases, Web Development, and more

### 👥 Community Features
- **Q&A Forum** - Ask questions and get help from peers
- **Resource Sharing** - Collaborate and share learning materials
- **Helpful Answers** - Mark useful contributions and build reputation
- **Discussion Threads** - Engage in focused academic discussions

### 🏆 Gamification & Progress Tracking
- **Learning Streaks** - Maintain consistency with daily learning streaks
- **Achievement Badges** - Unlock 10+ achievements through learning milestones
- **Points System** - Earn points for contributions and activity
- **Progress Calendar** - Visual representation of your learning activity

### 🎨 Modern UI/UX
- **Responsive Design** - Works seamlessly on desktop, tablet, and mobile
- **Dark Theme** - Eye-friendly dark interface with gradients
- **Smooth Animations** - GSAP-powered transitions and interactions
- **Accessibility** - Built with accessibility best practices

---

## 🚀 Usage

### Main Features

1. **Sign Up** - Create an account with your academic email
2. **Dashboard** - Access all study resources
3. **Upload** - Share documents with the community
4. **Community** - Ask questions and help others
5. **Chat** - Get help from AI assistant
6. **Streaks** - Track your learning consistency
7. **Achievements** - Unlock badges and earn points
